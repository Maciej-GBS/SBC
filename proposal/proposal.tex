\section{Viabilidad}

\noindent Hay cuatro dimensiones de viabilidad del sistema:
\begin{enumerate}
	\item Plausibilidad
	\item Justificación
	\item Adecuación
	\item Éxito
\end{enumerate}

\noindent Cada dimensión consiste de las características de tres categorías:
\begin{enumerate}
	\item Directivos y usuarios (DU)
	\item Los expertos (EX)
	\item La tarea (TA)
\end{enumerate}
